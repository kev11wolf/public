HTML, CSS, and JavaScript 101: Beginner's Guide

Welcome to this comprehensive guide designed for absolute beginners. We'll start from the basics and build up your skills step by step.

Introduction to Web Development

Web development involves creating websites using three core technologies: HTML for structure, CSS for styling, and JavaScript for interactivity.

Why Learn These?

HTML, CSS, and JS are the building blocks of the web. Mastering them opens doors to creating your own sites and understanding how the internet works.

Visual: Browser rendering HTML/CSS/JS layers

HTML Basics

HTML (HyperText Markup Language) is used to create the structure of a web page.

Step 1: Basic HTML Structure

Every HTML document starts with a doctype and includes html, head, and body tags.

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>My Page</title>
</head>
<body>
    <h1>Hello, World!</h1>
</body>
</html>
Tutorial: Create Your First HTML File
  1. Open a text editor (like Notepad).
  2. Copy the code above.
  3. Save as index.html.
  4. Open in a browser.

Common HTML Elements

Headings: Use <h1> to <h6> for titles.

<h1>Main Title</h1>
<h2>Subtitle</h2>

Paragraphs and Links:

<p>This is a paragraph.</p>
<a href="https://example.com">Link</a>

Images:

<img src="image.jpg" alt="Description">

Lists:

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

Forms

Basic form example:

<form>
    <input type="text" placeholder="Name">
    <button type="submit">Submit</button>
</form>

CSS Basics

CSS (Cascading Style Sheets) styles your HTML elements.

Adding CSS

Inline, internal, or external. We'll use internal for this guide.

<style>
body { background-color: lightblue; }
</style>

Selectors and Properties

Select by tag, class, or ID.

p { color: red; }
.class-name { font-size: 20px; }
#id-name { background: yellow; }
Tutorial: Style a Paragraph

Add the CSS above to your HTML and see the changes.

Box Model

Every element is a box with content, padding, border, margin.

Visual: Box Model Diagram
div {
    padding: 10px;
    border: 1px solid black;
    margin: 20px;
}

Layout: Flexbox

.container {
    display: flex;
    justify-content: space-between;
}

Responsive Design

Use media queries.

@media (max-width: 600px) {
    body { font-size: 14px; }
}

JavaScript Basics

JavaScript adds interactivity to your pages.

Adding JS

<script>
alert('Hello, World!');
</script>

Variables and Data Types

let name = 'John'; // String
const age = 30; // Number
let isStudent = true; // Boolean

Functions

function greet() {
    console.log('Hello!');
}
greet();

Add to your script and check the console.

DOM Manipulation

Change HTML with JS.

document.getElementById('demo').innerHTML = 'Changed!';

Events

Respond to user actions.

    alert('Clicked!');
});

Best Practices

  • Use semantic HTML (e.g., <header>, <nav>).
  • Keep CSS modular with classes.
  • Write clean, commented JS.
  • Test on multiple devices.

Conclusion

You've covered the basics! Practice by building small projects.
