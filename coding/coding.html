Welcome

1. Welcome to Web Development

Hello, aspiring web developer! In this comprehensive guide, you’ll learn the essentials of HTML, CSS, and JavaScript. We’ll build fun projects along the way.

By the end of this course, you’ll have built a multi-section, professional personal website that you can show off as part of your portfolio, plus mini-projects to demonstrate essential skills. Let’s get started!

Why learn these three technologies first?

HTML, CSS, and JavaScript form the foundational triad of modern web development. Mastering them is crucial to building any website or web app, from the simplest page to the most complex application.

Common career paths

Front-end developer, full-stack developer, UX/UI engineer, technical writer, DevOps – the possibilities are endless once you master these core skills and branch out into frameworks and more advanced technologies.

2. Setting Up Your Workspace

Proper tooling makes your coding journey smoother. While you can use any text editor, Visual Studio Code (VS Code) is popular for its community ecosystem and built-in features.

Installing VS Code

Download it from code.visualstudio.com. Then install the "Live Server" extension for real-time page reloading.

Browser Dev Tools

Every modern browser has built-in developer tools. Press F12 or Ctrl+Shift+I (Win) / Cmd+Option+I (Mac) to open them.

8. Intermediate JavaScript

In this section, we'll cover scope, closures, the this keyword, async/await, and localStorage.

Closures

A closure is the combination of a function bundled together (enclosed) with references to its surrounding state (the lexical environment). 


function makeCounter() {
  let count = 0;
  return function() {
    count++;
    return count;
  };
}

const counter = makeCounter();
console.log(counter()); // 1
console.log(counter()); // 2

Example: Todo List with LocalStorage 

<div id="todo-app">
  <input type="text" id="todo-input" />
  <button id="add-todo">Add</button>
  <ul id="todo-list"></ul>
</div>

<script>
  const input = document.getElementById('todo-input');
  const addBtn = document.getElementById('add-todo');
  const list = document.getElementById('todo-list');

  let todos = JSON.parse(localStorage.getItem('todos')) || [];

  function saveTodos() {
    localStorage.setItem('todos', JSON.stringify(todos));
  }

  function render() {
    list.innerHTML = '';
    todos.forEach((todo, index) => {
      const li = document.createElement('li');
      li.textContent = todo;
      li.addEventListener('click', () => {
        todos.splice(index, 1);
        saveTodos();
        render();
      });
      list.appendChild(li);
    });
  }

  addBtn.addEventListener('click', () => {
    if(input.value.trim() !== '') {
      todos.push(input.value.trim());
      input.value = '';
      saveTodos();
      render();
    }
  });

  render();
</script>
